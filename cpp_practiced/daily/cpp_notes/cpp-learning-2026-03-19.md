## C++ Concepts Covered (Lesson Progression)

### Basics (Lessons 1.1 - 1.5)
- Hello World / printing to screen
- Variables and data types (int, double, string, char)
- Basic calculator
- Variable memory concepts (declaration vs reassignment)

### Control Flow (Lessons 1.15 - 1.26)
- If statements, if-else
- While loops, for loops, do-while loops
- Sentinel-controlled programs
- Switch statements
- Logical operators (AND, OR, NOT)
- Random number generation

### Functions (Lessons 1.10, 1.27 - 1.30)
- Functions with multiple parameters
- Default arguments
- Scope resolution operator (::)
- Function overloading
- Recursion

### Object-Oriented Programming (Lessons 1.11 - 1.50)
- Classes and objects
- Getters and setters
- Constructors and destructors
- Member initializers
- Composition (has-a relationship)
- Friend functions
- Operator overloading
- The `this` keyword
- Inheritance (parent/child classes)
- Protected members
- Derived class constructors/destructors

### Advanced Topics (Lessons 1.51 - 1.60)
- Polymorphism
- Virtual functions
- Abstract classes and pure virtual functions
- Function templates
- Class templates
- Template specialization

### Data Structures (Lessons 1.31 - 1.40)
- Arrays (single and multi-dimensional)
- Passing arrays to functions
- Pointers
- Pass-by-reference with pointers
- sizeof operator
- Pointer arithmetic

## Tutorial Structure Patterns

### Each Lesson Contains:
1. **.sln file** - Visual Studio solution file
2. **Project folder** with:
   - .vcxproj (project file)
   - .vcxproj.filters
   - main.cpp (the actual code)
   - x64 folder (compiled binaries)

### Code Style:
- Heavily commented (explains every line)
- Includes learning tips in comments
- Clear variable names
- "Key takeaway" summaries at the end

### Naming Convention:
- Lessons numbered sequentially (1.1, 1.2, ... 1.60)
- AI source indicated in name (Claude, ChatGPT, Gemini, Bucky)
- Topics clearly labeled in folder names
